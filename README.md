1. Iris Flower Classification
This project is a Machine Learning classification project developed as part of the CodSoft Internship.

2. Objective
The objective of this project is to classify Iris flowers into different species based on their sepal and petal measurements.

3. Dataset
The dataset used in this project contains the following features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species

3.Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

4. Machine Learning Algorithm Used
- K-Nearest Neighbors (KNN)

5. Steps Performed
1. Imported required libraries
2. Loaded the dataset
3. Cleaned missing values
4. Visualized the data
5. Split the dataset into training and testing sets
6. Trained the KNN model
7. Evaluated the model using accuracy, confusion matrix, and classification report
8. Tested the model with new flower input values

6. Output
The model predicts the species of Iris flowers based on the input measurements.

7. Files in this Repository
- iris_flower_classification.ipynb → Colab notebook
- IRIS1.csv → Dataset file
- README.md → Project description
