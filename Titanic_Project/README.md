Titanic Project Files
1. Titanic Survival Prediction
This project is a Machine Learning classification project developed as part of the CodSoft Internship.

2. Objective
The objective of this project is to predict whether a passenger survived or not based on features like age, gender, class, and fare.

3. Dataset
The dataset used in this project contains the following features:
- Passenger Class (Pclass)
- Sex
- Age
- Number of Siblings/Spouse (SibSp)
- Number of Parents/Children (Parch)
- Fare
- Embarked
- Survived (Target)

4. Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

5. Machine Learning Algorithm Used
- Logistic Regression

6. Steps Performed
1. Imported required libraries
2. Loaded the dataset
3. Removed unnecessary columns (PassengerId, Name, Ticket, Cabin)
4. Handled missing values in Age and Embarked
5. Converted categorical data (Sex, Embarked) into numerical form
6. Visualized the data using graphs
7. Split the dataset into training and testing sets
8. Trained the Logistic Regression model
9. Evaluated the model using accuracy, confusion matrix, and classification report
10. Tested the model with new passenger input values

7. Output
The model predicts whether a passenger would survive or not based on the given input features.
